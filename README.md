# 🛒 TechStore Pro

A modern e-commerce web application built with React.

## 🚀 Live Demo

git add .
git commit -m "Added live demo link"
git push

## 📸 Features

- Browse products from a live API
- Search and filter products
- Product detail pages
- Add to Cart with quantity tracking
- Wishlist with heart toggle
- Categories page
- Login & Register with form validation
- Contact page
- Responsive design

## 🛠️ Built With

- React
- React Router DOM
- Vite
- FakeStore API

## 📦 Run Locally

git clone https://github.com/YOUR_USERNAME/techstore-pro.git
cd techstore-pro
npm install
npm run dev
